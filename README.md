# PyNPS
국민연금공단(NPS) 오픈 데이터를 활용하여 기업별 추정 연봉을 분석하고 동종업계 수준과 비교하는 데이터 대시보드.
정제되지 않은 2025년 12월 국민연금공단 Raw Data를 Pandas를 활용해 직접 전처리하고, 기업의 연봉 수준을 역산하여 시각화한 프로젝트입니다. 
단순한 데이터 출력을 넘어, 결측치 처리, 정규표현식을 이용한 텍스트 정제, 그룹화(Groupby)를 통한 통계 분석 등 실전 데이터 엔지니어링 역량을 위해 구축되었습니다.

## 기술 스택
* **Language:** Python 3
* **Data Engineering:** Pandas
* **Web Framework:** Streamlit
* **Data Visualization:** Plotly

## 주요 기능
- 정규표현식을 활용한 법인명 텍스트 정제 및 무의미한 결측치 제거 파이프라인 구축.
- 국민연금 요율(9%)을 역산하여 해당 기업의 1인당 월평균 급여 및 연봉 추정.
- 검색된 기업의 업종 코드를 기반으로, 동종업계 데이터만 추출 및 평균 급여를 산출하여 해당 기업과의 격차 비교.
- Plotly와 Streamlit Metric을 활용하여 데이터의 직관성과 가독성을 높힌 UI 제공.
